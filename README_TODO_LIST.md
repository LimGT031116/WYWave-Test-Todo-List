# Todo List Web Application

## Features
- Add a todo
- Delete a todo
- Update a todo
- Search todos by title, priority, or status

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **CSS Framework:** Bootstrap

## Installation & Running
- Clone this project from GitHub and open `index.html` in your browser to run the app.

## Usage
- Open index.html in your browser.
- Click Add Todo to go to the add page and create a new todo.
- You can return to the main page by clicking the Return button without adding a todo, or after adding a todo you will automatically return.
- The todo will appear in the list with its status, title, description, and priority.
- Use the Delete and Edit actions for each todo.
- Click the status checkbox to update the todo’s status.
- Click Edit to modify todo details; you will go to the edit page and return when done or click the Return button.
- On the main page, use the search and filter functions to find todos easily.
